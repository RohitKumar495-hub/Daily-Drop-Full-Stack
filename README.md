## 📌 Overview

**Daily Drop** is a modern e-commerce application designed to provide users with a smooth and engaging shopping experience.

It includes:

* 🛒 Product browsing & filtering
* 💳 Secure online payments
* 📦 Order management
* 📩 Email notifications
* ✨ Smooth animations for better UX

👉 Built with scalability and real-world use cases in mind.

---

## ✨ Key Features

### 👤 User Authentication

* Secure login/signup system
* User session management

### 🛍️ Product Management

* Browse products with dynamic UI
* Category-based filtering
* Product detail pages

### 🛒 Cart & Checkout

* Add/remove items from cart
* Real-time cart updates
* Smooth checkout flow

### 💳 Payment Integration

* Stripe payment gateway integration
* Secure transaction handling

### 📩 Email Notifications

* Automated communication system

* Clean and interactive interface

* Image upload and optimization via Cloudinary

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash id="f8j1wl"
git clone https://github.com/RohitKumar495-hub/daily-drop.git
cd daily-drop
```

### 2️⃣ Install dependencies

```bash id="3o1qzz"
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file and add:

```env id="d5s7yu"
MONGO_URI=your_mongodb_connection
STRIPE_SECRET_KEY=your_stripe_key
EMAILJS_PUBLIC_KEY=your_key
CLOUDINARY_API_KEY=your_key
CLOUDINARY_SECRET=your_secret
JWT_SECRET=your_secret
```

### 4️⃣ Run the project

```bash id="tt1gdf"
npm run dev
```
